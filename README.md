# AssignMate - A Flutter Application

Welcome to AssignMate, a Flutter application designed to help students manage their 
assignments and collaborate with their peers. AssignMate is a feature-rich app that 
provides students with all the tools they need to stay on top of their work, get it 
done ahead of time, and work effectively with their peers.

AssignMate can be downloaded from:
https://drive.google.com/file/d/1L6EAE3Q5oLcl0h0zWu5iu3CTKleK-bTV/view?usp=sharing
